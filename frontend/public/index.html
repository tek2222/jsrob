JSROB - Robot Visualization Tools

URDF Viewer

Interactive 3D visualization tool for URDF robot models.

  • Load and visualize URDF robot models
  • Support for COLLADA (.dae) and STL mesh files
  • Interactive 3D viewing with pan, zoom, and rotate
  • Joint animation capabilities
  • Real-time debug console
  • Tree structure visualization
Launch URDF Viewer

STL Viewer

Dedicated viewer for STL (STereoLithography) 3D model files.

  • Direct visualization of STL files
  • Support for both ASCII and binary STL formats
  • Interactive 3D manipulation
  • Model measurements and statistics
Launch STL Viewer

URDF Reader

Advanced URDF file parser and inspector.

  • Parse and validate URDF files
  • Inspect joint configurations
  • View robot structure hierarchy
  • Analyze link transformations
Launch URDF Reader

Getting Started

To use the URDF Viewer:

  1. Place your URDF files in the /public/models directory
  2. Place corresponding mesh files (.dae or .stl) in /public/models/meshes
  3. Launch the appropriate viewer using the buttons above
  4. Select your model from the dropdown menu
  5. Use mouse to interact with the 3D view:
    • Left click + drag to rotate
    • Mouse wheel to zoom
